# Tribeau 美容医療マーケット分析

トリビュー (tribeau.jp) の美容医療プラットフォームデータを解析したインタラクティブダッシュボード。

## ダッシュボード

- [マーケット分析](https://jim-auto.github.io/tribeau-analysis/) - クリニック分布・施術価格・地域比較
- [ユーザーガイド](https://jim-auto.github.io/tribeau-analysis/user.html) - 口コミ・満足度・施術詳細

## データ概要

- 全国4,886クリニック (東京1,805 / 大阪431 / 神奈川322)
- 200+施術の価格・口コミ統計
- 178,509件の口コミデータ分析（満足度あり116,151件）
- 30施術 × 全ページ取得（マルチスレッド並列収集）
- 12都道府県 × 15施術の価格マトリクス
